# 2041

This repo is configured to deploy the static site to GitHub Pages from the `work` branch.

## Deploy

1. Push to the `work` branch.
2. In GitHub, go to **Settings â†’ Pages** and ensure the source is set to **GitHub Actions**.
3. After the workflow completes, your site will be available at the Pages URL shown in the workflow output.

## Pages URL

Your GitHub Pages link will follow this pattern:

```
https://<github-username>.github.io/2041/
```

Replace `<github-username>` with your GitHub account or organization name.
